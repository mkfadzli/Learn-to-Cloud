# Fasa 0: Bermula dari sifar

> ⏱️ **Anggaran masa:** 3-4 minggu

Jika anda tiada pengalaman teknologi dan ingin menceburi bidang Kejuruteraan Cloud, fasa ini adalah untuk anda. Luangkan masa untuk memahami konsep-konsep asas ini — memahaminya dengan baik akan memudahkannya kemudian.

:::tip Sudah ada pengalaman teknologi?
Jika anda sudah berada dalam bidang teknologi dan ingin beralih ke cloud, sila ke [Fasa 1: Linux dan Bash](../phase1).
:::

---

## 🎯 Objektif

Di akhir fasa ini, anda akan:

- Memahami asas IT dan pengkomputeran cloud
- Mengetahui apa itu Linux, rangkaian, pengaturcaraan, dan DevOps secara keseluruhan
- Memutuskan sama ada anda komited untuk meneruskan kerjaya dalam kejuruteraan cloud
- Bersedia untuk pembelajaran berterusan dalam bidang yang sentiasa berkembang ini

## 🎓 Sijil (Pilihan)

**Piawaian industri untuk mereka yang ingin menceburi bidang teknologi ialah [CompTIA A+](https://www.comptia.org/certifications/a).**

| Soalan | Jawapan |
|--------|---------|
| Adakah ia diperlukan? | Tidak |
| Adakah ia membantu? | Bergantung pada gaya pembelajaran anda |
| Patutkah anda mengambilnya? | Selidik peranan di kawasan anda — jika mereka memintanya, pertimbangkan untuk mendapatkannya |
