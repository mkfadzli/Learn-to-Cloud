# Topik 2: Persediaan Cloud CLI

> ⏱️ **Anggaran masa:** 1-2 hari

Sekarang kita akan menyediakan persekitaran cloud anda. Pastikan anda telah mencipta akaun dengan mana-mana cloud yang kami sokong: AWS, Azure, atau GCP.

---

## 📚 Laluan Pembelajaran

1. **Buat akaun cloud:**
   - [AWS](https://aws.amazon.com/resources/create-account/)
   - [Azure](https://azure.microsoft.com/en-us/free/)
   - [GCP](https://cloud.google.com/free)

2. **Kenali console cloud:**
   - [AWS Management Console](https://aws.amazon.com/getting-started/hands-on/getting-started-with-aws-management-console/)
   - [Azure Portal](https://docs.microsoft.com/en-us/azure/azure-portal/azure-portal-overview)
   - [GCP Console](https://cloud.google.com/cloud-console)

3. **Belajari mengapa CLI lebih baik daripada klik:**
   - [Why use a CLI](https://www.freecodecamp.org/news/command-line-for-beginners/)

4. **Pasang CLI untuk pembekal yang anda pilih:**
   - [AWS CLI](https://aws.amazon.com/cli/)
   - [Azure CLI](https://docs.microsoft.com/cli/azure/install-azure-cli)
   - [GCP CLI](https://cloud.google.com/sdk/docs/install)

---
