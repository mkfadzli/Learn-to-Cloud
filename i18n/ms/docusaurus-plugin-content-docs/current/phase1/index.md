# Fasa 1: Linux dan Bash

> ⏱️ **Anggaran masa:** 2-3 minggu

Di Learn to Cloud, kami suka mendapat pengalaman praktikal secepat mungkin. Oleh itu, kami telah menyediakan makmal Capture The Flag (CTF) untuk anda berlatih Linux dan Bash. Sebelum anda boleh mengaksesnya, anda perlu meluangkan masa untuk belajar tentang makmal tersebut dan menyediakannya. Jangan risau — sepanjang proses ini, kami akan menyediakan petua, pautan dokumentasi, sumber, dan banyak lagi untuk membantu anda berjaya.

---

## 🎯 Objektif

Di akhir fasa ini, anda akan:

- Menyediakan komputer anda untuk semua makmal kami
- Mempelajari asas kawalan versi
- Mendapat kefamiliaran dengan cloud CLI
- Memahami konsep Infrastructure as Code
- Membina dan mengakses persekitaran makmal
- Menyelesaikan tujuh cabaran CTF untuk menguji kemahiran Linux anda

---

## 📋 Prasyarat

- Mac OS atau komputer berasaskan Ubuntu. Jika anda menggunakan Windows, anda akan menggunakan WSL.
