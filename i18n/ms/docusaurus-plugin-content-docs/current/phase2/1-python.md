# Topik 1: Asas Python

> ⏱️ **Anggaran masa:** 4 minggu (1 bab sehari)

Python ialah bahasa pengaturcaraan yang sangat popular dengan banyak penggunaan dalam kejuruteraan cloud dan mempunyai komuniti yang besar, sebab itulah kami memilihnya sebagai bahasa untuk dipelajari dalam panduan ini.

## 📚 Laluan Pembelajaran

Sertai buku [Python Crash Course](https://nostarch.com/pythoncrashcourse2e) oleh Eric Matthes. Ambil satu bab sehari dan anda sepatutnya dapat menyelesaikannya dalam sebulan dengan beberapa jam kajian sehari.

Buku ini adalah sumber yang hebat untuk pemula mempelajari pengaturcaraan Python. Ia merangkumi asas sintaks Python, struktur data, fungsi, dan modul. Buku ini juga menyertakan projek hands-on untuk membantu anda mempraktikkan kemahiran anda dan membina aplikasi dunia sebenar.

## 🧪 Uji Pengetahuan Anda

Setelah anda selesai dengan buku tersebut, tonton [kuliah CS50 tentang Python](https://youtu.be/LfaMVlDaQ24?feature=shared&t=49364). Semua di sini sepatutnya masuk akal bagi anda—jika tidak, anda belum bersedia untuk meneruskan.

### Latihan Berbantuan AI

Uji pengetahuan anda dengan pembantu AI menggunakan prompt ini:

1. Bolehkah anda bertanya kepada saya tentang perbezaan antara list dan tuple dalam Python?
2. Bolehkah anda kuiz saya tentang cara mencipta dictionary dalam Python?
3. Bolehkah anda meminta saya menjelaskan konsep list comprehension dengan contoh?
4. Bolehkah anda bertanya kepada saya apa itu lambda function dan bagaimana ia digunakan?
5. Bolehkah anda kuiz saya tentang cara mengendalikan exception dalam Python?
6. Bolehkah anda bertanya kepada saya tentang decorator Python dan bagaimana ia berfungsi?
7. Bolehkah anda meminta saya menjelaskan perbezaan antara method `__init__` dan `__new__` dalam class Python?
8. Bolehkah anda kuiz saya tentang cara membaca dan menulis fail dalam Python?
9. Bolehkah anda bertanya kepada saya tentang generator dalam Python dan bagaimana ia berbeza daripada fungsi biasa?
10. Bolehkah anda kuiz saya tentang cara menggunakan fungsi `map`, `filter`, dan `reduce` dalam Python?
