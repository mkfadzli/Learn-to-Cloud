> ⏱️ **Anggaran masa:** 2-3 hari

Dalam topik sebelumnya, buku yang disyorkan telah memperkenalkan anda kepada Django, jadi anda sepatutnya mempunyai konteks tentang cara REST API berfungsi. Sekarang mari kita dalami pengetahuan tersebut.

## 🧪 Uji Pengetahuan Anda

Uji pengetahuan anda dengan pembantu AI menggunakan prompt ini:

1. Bolehkah anda meminta saya jelaskan apa yang dimaksudkan dengan REST dan bagaimana ia berfungsi?
2. Bolehkah anda kuiz saya tentang perbezaan antara REST API dan SOAP API?
3. Bolehkah anda meminta saya jelaskan konsep resource dalam REST?
4. Bolehkah anda meminta saya jelaskan perbezaan antara request PUT dan POST dalam REST?
5. Bolehkah anda kuiz saya tentang cara menggunakan method HTTP GET, POST, PUT, dan DELETE dalam REST?
6. Bolehkah anda meminta saya jelaskan konsep status code dalam REST dan berikan contoh?
