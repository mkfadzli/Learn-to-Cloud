# Topik 4: Pangkalan Data

> ⏱️ **Anggaran masa:** 3-4 hari

Kami akan menggunakan pangkalan data untuk menyimpan data aplikasi kami. Untuk ini, kami perlu memahami terlebih dahulu bagaimana pangkalan data berfungsi dan cara berinteraksi dengannya.

1. **Kaji asas SQL:** [Intro to SQL](https://www.khanacademy.org/computing/computer-programming/sql)

   | Cloud | Pangkalan Data | Tutorial |
   |-------|----------------|----------|
   | AWS | DynamoDB | [DynamoDB Tutorial For Beginners](https://www.youtube.com/watch?v=2k2GINpO308) |
   | Azure | Cosmos DB | [Cosmos DB Essentials Playlist](https://youtube.com/playlist?list=PLLasX02E8BPDd2fKwLCHnmWoyo4bL-oKr&feature=shared/) |

3. **Selesaikan quickstart Python untuk pangkalan data pilihan anda:**
   - [AWS DynamoDB Python Quickstart](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/example_dynamodb_Scenario_GettingStartedMovies_section.html)
