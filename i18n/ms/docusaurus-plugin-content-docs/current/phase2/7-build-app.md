## 📚 Laluan Pembelajaran

### Prasyarat

Projek ini menggunakan **Dev Containers** untuk menyediakan persekitaran pembangunan yang konsisten. Sebelum memulakan:

### Alat yang Diperlukan

1. **Docker Desktop** - [Muat turun di sini](https://www.docker.com/products/docker-desktop/)
   - Dev Containers berjalan di dalam Docker, jadi anda perlu memasangnya dan menjalankannya

2. **VS Code** dengan **sambungan Dev Containers**
   - Pasang [sambungan Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)
   - Ini membolehkan VS Code membuka projek di dalam container dengan semua dependensi telah dikonfigurasi

### Apa itu Dev Containers?

Dev Containers membolehkan anda membangun di dalam Docker container yang mempunyai semua alat dan dependensi yang telah dipasang. Apabila anda membuka repo starter dalam VS Code, anda akan melihat prompt untuk "Reopen in Container" - klik ia! Ini memastikan semua orang mempunyai versi Python, pakej, dan konfigurasi yang sama.

Kami mengesyorkan anda melihat sumber-sumber ini untuk mengenali Dev Containers:

- [Dev Containers in VS Code](https://www.youtube.com/watch?v=b1RavPr_878) - Pengenalan pantas (5 min)
- [Dokumentasi Dev Containers](https://code.visualstudio.com/docs/devcontainers/containers) - Dokumentasi rasmi
