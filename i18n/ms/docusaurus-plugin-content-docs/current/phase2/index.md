# Fasa 2: Pengaturcaraan & Integrasi AI

> ⏱️ **Anggaran masa:** 4-5 minggu

Selamat datang ke Fasa 2! Fasa ini sepenuhnya mengenai pengaturcaraan dengan Python dan mengintegrasikan keupayaan AI ke dalam aplikasi anda.

Pengaturcaraan adalah kemahiran asas untuk kejuruteraan cloud, membolehkan anda mencipta, mengurus, dan mengoptimumkan sumber cloud dengan cekap. Tambahan pula, mengintegrasikan AI dan Large Language Models (LLMs) ke dalam aplikasi semakin menjadi kemahiran penting bagi jurutera cloud moden.

Anda tidak perlu tahu bagaimana untuk membina UI tetapi anda perlu tahu cara bekerja dengan API dan perkhidmatan cloud lain seperti storan, pangkalan data, dan perkhidmatan AI. Mengetahui cara mengaturcara dan mengintegrasikan AI akan menjadikan anda lebih kompetitif dalam pasaran kerja.

:::warning Penting
Apabila peserta menyelesaikan fasa ini, ramai yang akan berkongsi kod mereka. Kami mengesyorkan anda **jangan salin dan tampal** penyelesaian sesiapa sahaja. Anda hanya merugikan diri sendiri dan tidak akan berjaya dalam bidang ini jika itu adalah mentaliti yang anda ada.

Selain itu, kami mengesyorkan anda mematikan sebarang pembantu AI untuk pelengkapan kod, walaupun menggunakan AI untuk bertanya penjelasan dan penjelasan adalah sangat digalakkan.
:::
