# Topik 5: Deployment dan Konfigurasi Pangkalan Data

> ⏱️ **Anggaran masa:** 2-3 hari

---

## 📚 Laluan Pembelajaran

- Kaji

  - [Asas PostgreSQL](https://www.prisma.io/dataguide/postgresql/getting-to-know-postgresql) dan [Fundamental](https://www.postgresql.org/docs/online-resources/): Memahami seni bina PostgreSQL, termasuk jadual, skema, dan pengguna.
  - Pelajari cara memasang PostgreSQL pada VM: [AWS](https://www.youtube.com/watch?v=KQ4eIv6h7Sk), [Azure](https://www.youtube.com/watch?v=_f1gKlV60Iw) dan [GCP](https://dev.to/vishalpaalakurthi/how-to-set-up-a-postgresql-server-on-a-virtual-machine-473f)
  - Kawalan Akses & Autentikasi: Pelajari cara menggunakan **pg_hba.conf** untuk menguruskan autentikasi dan menguatkuasakan akses pangkalan data yang selamat.

---

## 🧪 Uji Pengetahuan Anda

Uji pengetahuan anda dengan pembantu AI, berikut adalah beberapa contoh prompt untuk menguji pengetahuan anda tentang deployment pangkalan data relasional dan konfigurasinya:

1. Jelaskan komponen utama seni bina PostgreSQL. Bagaimanakah jadual, skema, dan peranan berinteraksi?
2. Apakah langkah-langkah untuk memasang dan mengkonfigurasi PostgreSQL pada VM berasaskan Linux?
3. Bagaimanakah anda akan mengkonfigurasi PostgreSQL dengan selamat untuk akses jarak jauh dalam subnet persendirian?
4. Apakah tujuan `pg_hba.conf`, dan bagaimana ia mengawal autentikasi dalam PostgreSQL?
5. Jelaskan cara mencipta dan menguruskan pengguna serta peranan PostgreSQL dengan keizinan yang sesuai.
