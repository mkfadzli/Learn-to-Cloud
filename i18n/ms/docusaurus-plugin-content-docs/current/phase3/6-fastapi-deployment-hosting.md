# Topik 6: Deployment FastAPI dan Pengehosan API

> ⏱️ **Anggaran masa:** 2-3 hari

Dengan pengetahuan terdahulu anda tentang FastAPI dari fasa sebelumnya, kini anda akan memberi tumpuan kepada deployment aplikasi FastAPI tersebut pada VM berasaskan cloud.

---

## 📚 Laluan Pembelajaran

- Kaji: cara deploy aplikasi FastAPI menggunakan [AWS EC2](https://www.youtube.com/watch?v=SgSnz7kW-Ko), [Azure Virtual Machines](https://medium.com/%40swamynathanayyappan/how-to-deploy-a-fast-api-python-web-application-to-azure-virtual-machine-c5eb84284a81), atau [Google Compute Engine](https://www.youtube.com/watch?v=Lcuas-20g1g).

---

## 🧪 Uji Pengetahuan Anda

1. Bagaimanakah anda menyimpan dan menguruskan pemboleh ubah persekitaran dengan selamat pada VM cloud untuk FastAPI?
2. Bagaimanakah anda akan menyediakan peraturan firewall atau security group untuk membenarkan trafik HTTP luaran ke aplikasi FastAPI anda?

---

## 🛠️ Tugas Hands-on

Deploy aplikasi FastAPI pada VM berasaskan cloud dalam Subnet Awam rangkaian cloud anda.

---

## ✅ Senarai Semak Topik

Sebelum meneruskan, pastikan anda telah:

- [ ] Mengkaji sumber deployment FastAPI untuk penyedia cloud anda
- [ ] Memahami cara menguruskan pemboleh ubah persekitaran dengan selamat
- [ ] Mengkonfigurasi security group/peraturan firewall untuk trafik HTTP
- [ ] Deploy FastAPI pada VM di subnet awam
- [ ] Mengesahkan API boleh diakses dari internet
