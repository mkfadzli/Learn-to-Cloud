# Fasa 3: Asas Platform Cloud

> ⏱️ **Anggaran masa:** 4-5 minggu

Selamat datang ke Fasa 3! Fasa ini memberi tumpuan kepada asas platform cloud - konsep dan kemahiran teras yang anda perlukan untuk bekerja dengan perkhidmatan cloud secara berkesan. Anda akan mempelajari semuanya dari mesin maya dan rangkaian sehinggalah kepada keselamatan dan penggunaan aplikasi. Melalui latihan hands-on dan projek dunia sebenar, anda akan membina pengalaman praktikal yang diperlukan untuk bekerja dengan platform cloud secara profesional.

---

## 🎯 Objektif

Pada akhir fasa ini, anda akan:

- Memahami perkhidmatan dan infrastruktur platform cloud teras
- Deploy dan mengurus mesin maya dan sumber pengkomputeran
- Konfigurasi rangkaian yang selamat dan kawalan akses
- Sediakan dan uruskan pangkalan data dan aplikasi API
- Selesaikan projek hands-on yang menunjukkan kemahiran cloud
- Bersedia untuk pensijilan cloud peringkat bersekutu

---

## 📋 Prasyarat

- Menyiapkan Fasa 1: Linux dan Bash
- Menyiapkan Fasa 2: Pengaturcaraan & Integrasi AI
- Asas Git dan kawalan versi
- Akaun platform cloud (AWS, Azure, atau GCP)
