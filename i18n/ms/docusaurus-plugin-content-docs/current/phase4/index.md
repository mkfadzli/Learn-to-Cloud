# Fasa 4: Asas DevOps

> ⏱️ **Anggaran masa:** 4-5 minggu

Selamat datang ke Fasa 4! Fasa ini merangkumi asas DevOps - amalan dan alat yang membolehkan pasukan menyampaikan perisian dengan lebih cepat dan boleh dipercayai. Anda akan belajar tentang pemkontaineran, talian paip CI/CD, Infrastruktur sebagai Kod, dan pemantauan.

---

## 🎯 Objektif

Di akhir fasa ini, anda akan:

- Memahami amalan teras DevOps dan manfaatnya
- Melaksanakan talian paip CI/CD untuk pengujian dan penggunaan automatik
- Membina infrastruktur sebagai kod menggunakan Terraform
- Membina dan menguruskan kontena dengan Docker
- Menyediakan pemantauan dan kebolehcerapan untuk aplikasi
- Menggunakan alat AI untuk mengoptimumkan tugas dan aliran kerja DevOps
- Menyiapkan projek capstone yang menunjukkan kemahiran DevOps
